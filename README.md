# Machine Learning Evaluation Metrics

This repository demonstrates the manual and programmatic calculation of common **Machine Learning model evaluation metrics** using small manually created datasets.

The repository covers evaluation metrics for both **Classification** and **Regression** models.

---

##  Classification Metrics

The following classification metrics are implemented and explained:

- Accuracy
- Precision
- Recall
- F1 Score
- Specificity
- ROC Curve
- AUC

Dataset used: A manually created **Loan Approval dataset**

---

##  Regression Metrics

The following regression metrics are implemented and explained:

- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)
- Root Mean Squared Error (RMSE)
- Mean Absolute Percentage Error (MAPE)
- R² Score
- Adjusted R²

Dataset used: A manually created dataset for **score prediction based on study hours**.

---

##  Purpose

The goal of this repository is to:

- Understand **how evaluation metrics work mathematically**
- Perform **manual calculations of metrics**
- Verify results using **Python (NumPy, scikit-learn)**
- Provide a clear learning reference for **Machine Learning beginners**
